# Evaluation

Evaluate the quality and consistency of your agentic system outputs.

## Evaluation Process

1. **Configuration**: Define the evaluator settings.
2. **Evaluators**: Create specific evaluators for different tasks.
3. **Datasets**: Define test cases with inputs and expected outputs.
4. **Run**: Execute evaluations and analyze results.

Evaluation helps ensure that changes to prompts or tools don't negatively impact the agent's performance.
