# Workflow Examples

Neuron provides several examples of complex agentic workflows.

## Deep Research Agent

An implementation that uses loops and branches to iteratively gather information, analyze it, and refine its search until a comprehensive report is generated.

## Travel Agent (Laravel)

A practical demo of buildling a multi-agent system in Laravel, coordinating between flight search, hotel booking, and activity planning agents.

## Human-in-the-Loop with Checkpoints

Demonstrates how to use interruptions and persistence to manage tasks that require human oversight, such as financial transactions or system modifications.

Find the full source code for these examples on [GitHub](https://github.com/neuron-core/neuron-ai/tree/main/examples).
